# Name List Printer

This Python script takes user input for 10 names and stores them in a list. It then prints each name with title case formatting.

## Usage

1. Run the script.
2. Enter 10 names, one at a time, when prompted.
3. The script will store the names in a list and print them with title case formatting.

## Example
```
Input:
John
jane DOE
michael Jackson
Sarah Smith
peter parker
emma Watson
James Bond
lucy liu
robert smith
emily brown
```
```
Output:
John
Jane Doe
Michael Jackson
Sarah Smith
Peter Parker
Emma Watson
James Bond
Lucy Liu
Robert Smith
Emily Brown
```
